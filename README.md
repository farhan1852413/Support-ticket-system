# 🎫 AI-Powered Support Ticket System

A modern full-stack support ticket management platform with AI integration, smart ticket handling, professional dashboards, and responsive UI.

---

## ✨ Features

- 🤖 Google Gemini AI Integration
- 🎨 Modern Responsive UI
- 🔐 JWT Authentication & Authorization
- 📊 Dashboard Analytics
- 📝 Smart Ticket Management
- 👤 User Profiles & Activity Tracking
- 🏆 Achievement System
- ⚡ Fast React + Vite Frontend
- 🛠️ Node.js + Express Backend
- 🗄️ MySQL Database

---
